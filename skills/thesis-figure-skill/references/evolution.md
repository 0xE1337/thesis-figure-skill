# 自进化基线 — 已验证最佳实践参数

> **原则：标准只升不降。** 本文件记录的参数都经过实际渲染验证，是当前已知的最优值。
> 下次画同类图时**直接使用这些参数作为起点**，不要从零试错。
>
> **何时加载**：步骤②加载专项规则时，同时加载本文件。
> **何时更新**：步骤⑦经验沉淀时，如果发现更优参数，更新本文件（只更新为更好的值，不降级）。

---

## 通用基线（所有 TikZ 图适用）

### 间距参数

| 参数 | 基线值 | 来源 | 说明 |
|------|--------|------|------|
| 同层相邻框 x 间距 | ≥ 1.5cm | batch10-25 验证 | < 1.0cm 必重叠 |
| 上下相邻框 y 间距 | ≥ 1.2cm | batch8 验证 | 用 `below=1.2cm of` 不用 `0.3cm` |
| zone 到最外层内容 padding | 0.8-1.0cm | batch25 验证 | > 1.5cm 浪费空间, < 0.5cm 虚线切文字 |
| zone 标题与首行内容间距 | ≥ 0.8cm | batch25 rule#42 | 标题框~0.4cm + 空白~0.4cm |
| 图例框与最近内容 | ≥ 0.5cm | batch10 验证 | 图例放在所有 zone 外部下方居中 |

### 箭头参数

| 参数 | 基线值 | 来源 | 说明 |
|------|--------|------|------|
| Stealth scale（箭身≥2cm） | ≤ 1.2 | batch25 rule#35 | 大于1.2箭头显粗 |
| Stealth scale（箭身1-2cm） | ≤ 0.9 | batch25 验证 | |
| Stealth scale（箭身0.5-1cm） | ≤ 0.7 | batch25 验证 | |
| 弯折点到目标框距离 | ≥ 1.5cm | batch18 验证 | rounded corners 吃掉 ~0.6cm |
| 树状扇出横线到目标框 | ≥ 1.5cm | batch12 验证 | 太近则"刚拐过来就到了" |
| 小型汇合节点两侧间隙 | ≥ 0.4cm | batch25 rule#37 | ⊕/⊗ 需要可见箭身 |

### 文字参数

| 参数 | 基线值 | 来源 | 说明 |
|------|--------|------|------|
| 最小可读字号 | `\scriptsize` | 300dpi 验证 | 比这小在 PNG 中不可读 |
| 框内文字 | `\footnotesize` | 通用 | 框有空间可用 `\small` |
| 标题文字 | `\small\bfseries` | 通用 | |
| 标签背景 | 默认透明 | batch7 验证 | 只在与生命线重叠时用 `fill=white, fill opacity=0.85` |

## 基线更新规则

1. **只升不降**：如果新验证的参数比当前基线更优（更少重叠、更好对齐），更新基线值
2. **标注来源**：更新时标注 batch 编号和验证方式（渲染确认 / 用户认可 / 验证器通过）
3. **退化警告**：如果生成的图没有使用基线参数，在审查阶段标记为 WARN——"你为什么不用已验证的基线值？"
4. **删除条件**：只有当参数被证明在某些场景下有害时才删除，而不是"没用到就删"
